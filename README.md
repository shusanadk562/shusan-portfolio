# Shusan-Portfolio
# 🚀 Personal Portfolio Website

A modern and responsive **personal portfolio website** built to showcase my skills, projects, and creative work. This portfolio highlights my development experience, design abilities, and completed projects through a clean and interactive user interface.

---

## 🌐 Live Demo

🔗 https://www.shusanadhikari.com.np/

---

## 📌 Features

* 🎨 **Modern UI Design**

  * Clean and professional layout
  * Well-balanced colors and typography
  * Smooth user experience

* 📂 **Projects Showcase**

  * Displays multiple projects with:

    * Title
    * Description
    * Tech stack
    * Live preview links
  * Organized in card-based layout

* 🛠️ **Skills Section**

  * Highlights technical skills and tools
  * Includes development and design technologies

* 📱 **Responsive Design**

  * Fully responsive across all devices
  * Optimized for mobile, tablet, and desktop

* ⚡ **Interactive UI**

  * Hover effects and transitions
  * Smooth animations

* 📞 **Contact Section**

  * Easy ways to connect
  * Clean and accessible layout

---

## 🛠️ Technologies Used

* **React (Vite)** – Frontend framework
* **JavaScript (ES6+)** – Logic and interactivity
* **Styled Components / CSS** – Styling and layout

---

## 🎯 Purpose

This portfolio was created to:

* Showcase my projects and skills
* Build a strong online presence
* Practice modern frontend development
* Demonstrate UI/UX design capabilities

## 📈 Key Highlights
